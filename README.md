# TFI-lisandro-cruzloresi-drone-roll
En este repo se encuentra la simulacion realizada para el trabajo final integrador de Lisandro Cruz Loresi.
El trabajo consiste en el control del Roll de un drone cuadricoptero. Al ejecutar el codigo ademas de mostrar las señales del angulo y el torque 
aplicado por las perturbaciones se corre una animacion para hacer el trabajo mas didactico.
